# 실행 방법
1. 우측 상단의 code ->DownloadZip 버튼을 통해 게임 파일을 다운로드 후 압축 해제를 합니다
2. 생성된 폴더 내의 main.py를 실행합니다.

  * image, bgm, rock 파일은 게임 그래픽요소입니다.
  
# 게임 소개
## 1. 게임 소개
이 게임은 운석 피하기 게임입니다. 날아오는 운석을 피하고 위험하면 워프를 사용하여 높은 점수를 획득하세요!

## 2. 조작법
우주선 움직이기 : 마우스 움직이기

일시정지: 'P'키

아이템-'워프'사용 : 마우스 아무버튼 클릭

## 3. 난이도 올라가는 기준
첫번째 상승 -> 점수: 300점

두번쨰 상승 -> 점수 : 400점

마지막 상승 -> 점수 : 500점
